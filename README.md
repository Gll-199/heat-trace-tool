# 电伴热外贸智能助手

专为电伴热行业外贸业务员设计的 AI 辅助工具。

## 使用方法

1. 打开 `index.html`（或访问在线版本）
2. 粘贴客户英文询盘，点击「解析询盘」
3. 确认参数后点击「生成回复」
4. 复制英文回复发给客户

## 功能

### 基础版（免费）
- 客户询盘智能解析
- 英文回复一键生成
- 中文内部备注清单
- 参数完整度检查
- 产品选型参考表

### PRO 版（¥19）
- 热损计算器
- 出口合规速查
- 持续功能更新

## 本地运行

```bash
node server.js
```

然后在浏览器打开 http://localhost:3000

## 部署到 GitHub Pages

1. 在 GitHub 创建一个新仓库
2. 把本目录所有文件上传到仓库
3. 进入仓库 Settings → Pages
4. 选择 main 分支，root 目录
5. 等待几分钟，访问 https://你的用户名.github.io/仓库名

## 技术
纯前端 HTML + CSS + JavaScript，零依赖，任何静态托管都可部署。
